# TED_20230228
  Enunciados:
      1) Crie uma função recursiva que exiba o resultado do fatorial de um número (Pela limitação da recursividade, o número de entrada deverá ser baixo para não dar estouro(limite de entrada = 12)): O código deve trazer como comentário a condição de parada e como escrever a função para o termo n em função do termo anterior.
      2) Crie uma função recursiva que exiba o total de elementos negativosde um vetor de inteiros, de N posições, passado como parâmetro. O código deve trazer como comentários a condição de parada e como escrever a função para o termo n em função do termo anterior.
      3) Crie uma função recursiva que exiba a quantidade de dígitos de um número inteiro passado como parâmetro. O código deve trazer como comentários a condição de parada e como escrever a função para o termo n em função do termo anterior.
      4) Crie uma função recursiva que exiba o resultado da inversão de uma cadeia de caracteres (Ex.: entrada = teste ; saída = etset). Deve se utilizar a função SUBSTRING da Java. O código deve trazer como comentários a condição de parada e como escrever a função para o termo n em função do termo anterior.
